# ouracli Homebrew tap

```sh
brew tap ouracli/oura
brew install oura
```

The `Formula/oura.rb` formula is generated and updated automatically by
[goreleaser](https://github.com/ouracli/oura/blob/main/.goreleaser.yaml) on
every tagged release of [ouracli/oura](https://github.com/ouracli/oura).

> Unofficial project — not affiliated with Oura Health Oy.
